# Credit Card Validator

## Описание
Программа на языке программирования Go, которая проверяет номера кредитных карт с помощью проверенного временем алгоритма Луна, определяет банк-эмитент. Список банков хранится в текстовом файле вида csv.

## Возможности
- загрузка данных из текстового файла
- валидация пользовательского ввода
- проверка номера карты алгоритмом Луна
- определение банка-эмитента по номеру карты

## Установка и запуск

Для работы приложения необходимо иметь установленный язык программирования go. Клонировать проект на устройство. Запустить через консоль программу с помощью:
```bash
go run main.go
```
